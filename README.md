# Gestor de Tareas Web

## Funcionalidades

- Crear tareas
- Visualizar tareas
- Editar tareas
- Eliminar tareas
- Marcar tareas como completadas o pendientes
- Persistencia usando Local Storage

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Uso de IA

Se utilizó ChatGPT de para:

- Generar reevisar estructura inicial del proyecto.
- Consultar buenas prácticas en el desarrollo del proyecto.
- Corregir errores de JavaScript y funcionamientos que no conocía.

El código fue revisado, probado y comprendido antes de su entrega.

## Ejecución

Abrir el archivo index.html en cualquier navegador moderno. Actualizado styless
